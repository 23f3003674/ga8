# Supply Chain Correlation Analysis — OptimalFlow Logistics

This repository contains the correlation analysis conducted for a major automotive manufacturer's supplier performance data.

**Contact:** 23f3003674@ds.study.iitm.ac.in

## Files
- `correlation.csv` — Pearson correlation matrix for the five supply chain metrics.
- `heatmap.png` — Heatmap screenshot (Excel-style color scale: red → white → green).
- `README.md` — This file.

## Notes
- Correlations computed on columns: Supplier_Lead_Time, Inventory_Levels, Order_Frequency, Delivery_Performance, Cost_Per_Unit.
- Heatmap generated with a diverging color scale centered at 0 (red = negative correlation, green = positive correlation).
- Image dimensions: 450x450 pixels (within required 400–512 px).
